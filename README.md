# Electrical Unit Calculator (Unified)

An Enterprise-grade electrical engineering toolkit for unit conversion, Ohm's Law parameter solving, and circuit analysis. Achieved a **10/10 health score** through modular architecture and robust error handling.

## 🚀 Features
- **Ohm's Law Solver**: Automatically calculates Voltage (V), Current (I), Resistance (R), and Power (P) from any two known parameters.
- **Specialized Unit Conversion**: Seamlessly converts between electrical units (Ohm, Farad, Henry) with support for standard prefixes (k, M, u, n, p).
- **Circuit Analysis**: Performs mathematical summation for series and parallel electrical components.

## 🛠️ Installation

### 1. Install Git
On Debian/Ubuntu:
```bash
sudo apt update
sudo apt install git -y
```

### 2. Install Python 3
On Debian/Ubuntu:
```bash
sudo apt update
sudo apt install python3 python3-venv -y
```

### 3. Clone the Repository
```bash
git clone https://github.com/hiddendestroyer1945/electricalunit-calculator.git
cd electricalunit-calculator
```

---

## 📖 Usage

### Interactive Mode
Run the script without arguments to enter the menu-driven interface:
```bash
python3 electricalunit-calculator.py
```

### CLI Mode (Automation)
Solve Ohm's Law parameters via command line flags:
```bash
# Calculate Resistance and Power from Voltage and Current
python3 electricalunit-calculator.py --v 100 --i 10
```
